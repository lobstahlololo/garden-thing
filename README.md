The Dynamic Micro-Climate Gardening Assistant is an interactive web-based interface that shifts standard, generalized gardening calendars into localized insights. It works by taking the base meteorological data of a specific region and modifying it through environmental variables unique to an individual yard—such as elevation dips, nearby heat sinks, wind barriers, and real-time weather fluctuations. By calculating how physical parameters shift ambient temperatures, the app ensures growers no longer plant too early or get caught unprepared by an irregular local freeze.
